## Responsive Behavior

### On Desktop (1200px+):
- All images display at full dimensions
- Hero background covers full width
- Team cards display in 3-column grid

### On Tablet (760px - 980px):
- Images scaled to fit container
- Blog section adjusts to single column
- Team cards may display in 2-column grid

### On Mobile (< 760px):
- Images scale down with container
- All sections stack vertically
- Team cards display in single column
- Touch-friendly sizing maintained

## Performance Optimization Tips

### For Background Images:
- Use JPEG format (best compression)
- Compress to 400-500KB maximum
- Consider WebP for 30% smaller files
- Use `cover` CSS property for proper scaling

### For Content Images:
- Use JPEG for photographs
- Use PNG for graphics/logos
- Keep under 150KB each
- Ensure 72+ DPI for web quality

### For Team Photos:
- Use JPEG for best quality at small size
- Keep under 100KB per photo
- Ensure consistent lighting/style
- Use white or neutral background
